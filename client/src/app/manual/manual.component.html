Manual estabelecimentos

Os estabelecimentos que desejarem ofertar serviços para atender as necessidades dos trabalhadores remotos poderão realizar a inscrição no site, na aba cadastre-se para receber a certificação. O manual com algumas recomendações encontra-se disponível abaixo para download.

Manual municípios

Os municípios que estiverem interessados em se cadastrar para receber os Nômades Digitais, poderão realizar o cadastro, na aba cadastre-se, para ter suas informações divulgadas no site. Um manual com algumas recomendações encontra-se disponível abaixo para download.
